# Group Chat Spider Fight - Public Site

Simple landing page with privacy policy and support pages for App Store submission.

## Setup Instructions

1. Create a new **public** repo on GitHub (e.g., `group-chat-spider-fight`)

2. Copy these files to the new repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/group-chat-spider-fight.git
   cd group-chat-spider-fight
   # Copy index.html, privacy.html, support.html here
   git add .
   git commit -m "Initial site files"
   git push
   ```

3. Enable GitHub Pages:
   - Go to repo Settings > Pages
   - Source: Deploy from a branch
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
   - Click Save

4. Your site will be available at:
   ```
   https://YOUR_USERNAME.github.io/group-chat-spider-fight/
   ```

5. Update App Store Connect with these URLs:
   - Privacy Policy: `https://YOUR_USERNAME.github.io/group-chat-spider-fight/privacy.html`
   - Support URL: `https://YOUR_USERNAME.github.io/group-chat-spider-fight/support.html`

## Files

- `index.html` - Simple landing page
- `privacy.html` - Privacy policy for App Store
- `support.html` - Support/FAQ page for App Store

## Customization

- Update email addresses (currently `spiderfight.app@gmail.com`)
- Adjust colors if branding changes
- Add App Store link once published
